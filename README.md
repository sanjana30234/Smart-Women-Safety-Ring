# Smart Women’s Safety Ring

An IoT-based wearable safety device designed to enhance personal security by sending instant emergency alerts with real-time location tracking.



##  Problem Statement
In emergency situations, it is often difficult to manually call for help. This project solves that by providing a quick SOS activation system in a wearable form.



##  Features
- SOS emergency alert system with single-button activation  
- Real-time GPS location tracking  
- GSM module for instant message/call alerts  
- Sends emergency notification to predefined contacts  
- Compact and wearable design for quick access  



##  Tech Stack / Components
- Arduino / Microcontroller (if used)  
- GPS Module  
- GSM Module  
- Embedded C / Arduino IDE  
- IoT Hardware Integration  



##  How It Works
1. User presses SOS button on the ring/device  
2. GPS captures current location  
3. GSM module sends alert SMS/call  
4. Emergency contacts receive location instantly  


## Impact
- Faster emergency response  
- Improved personal safety  
- Reliable alert system in critical situations
